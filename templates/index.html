What's this?

Sometimes you need to export a Trello board to a spreadsheet for further processing. This application will help you do that. A typical use case could be this scenario: Managment: -"I would like a report of project activity and status." You: -"Sure, here is a link to our Trello board", Management: -"What's Trello?".

Will you be able to read my Trello data?

Yes, for a short while when it is downloaded and transferred to a spreadsheet file. After it has been sent it will be deleted. Trello account access is only granted for an hour.

Can you edit my Trello data?

No, this application only requests read permissions for your Trello data for an hour.

I clicked 'export' but nothing happened?

Are you sure? Check your recently downoaded files. Sometimes the export process is really quick.

Is everything exported?

No, only basic information for each card like id, name, description etc.
